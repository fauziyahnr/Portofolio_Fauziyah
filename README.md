# Fauziyah's Portfolio

A modern, responsive, and professional portfolio website built for showcasing software engineering skills, projects, and experience.

## 🌟 Features

- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean and professional design with smooth animations
- **Sections Included**:
  - Hero section with call-to-action buttons
  - About me with statistics
  - Skills & technologies showcase
  - Featured projects with live demos and code links
  - Professional experience timeline
  - Blog section for sharing knowledge
  - Contact form and social media links
- **Performance Optimized**: Fast loading with lazy loading and optimized images
- **Accessibility**: Semantic HTML and WCAG compliance
- **SEO Friendly**: Meta tags and structured content

## 🚀 Getting Started

### Prerequisites
- A web browser (Chrome, Firefox, Safari, Edge)
- No build tools required - ready to use!

### Installation

1. Clone the repository:
```bash
git clone https://github.com/fauziyahnr/Portofolio_Fauziyah.git
cd Portofolio_Fauziyah
```

2. Open in a browser:
```bash
# Simply open the index.html file in your web browser
# Or use a local server for better performance
python -m http.server 8000
# Then visit http://localhost:8000
```

## 🎯 Customization

### Update Personal Information
Edit `index.html` to add your:
- Name and title
- Contact email
- Social media links
- Project details
- Work experience

### Customize Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;    /* Change primary color */
    --secondary-color: #ec4899;  /* Change secondary color */
    /* ... other variables ... */
}
```

### Add Projects
Add new project cards in the Projects section by duplicating the `.project-card` structure.

### Update Blog Posts
Add new blog articles by duplicating the `.blog-card` structure.

## 🚀 Deployment

### Deploy to GitHub Pages
1. Push to your GitHub repository
2. Go to Settings → Pages
3. Select source as `main` branch
4. Your portfolio will be live at `https://fauziyahnr.github.io/Portofolio_Fauziyah/`

### Deploy to Other Platforms
- **Netlify**: Drag and drop the folder
- **Vercel**: Connect your GitHub repository
- **Firebase**: Use Firebase hosting
- **Heroku**: Use static site builder

## 🐛 Troubleshooting

### Portfolio not displaying properly on mobile?
- Clear browser cache
- Check viewport meta tag in HTML
- Test on different mobile devices

### Images not loading?
- Verify image paths
- Use absolute URLs for external images
- Check image file permissions

### Contact form not working?
- Update email link with your email
- Set up backend for form submissions (optional)
- Test form validation in browser console
